# Printer Deployment Automation Tool

Python-based tool for automating printer connection and default printer setup in enterprise environments.

## Features
- Automatically connects network printers
- Sets default printers
- Reduces manual configuration work

## Tech Stack
- Python
- win32print
- tkinter

## Background
This tool was developed as part of IT infrastructure work in a company with 1000+ endpoints to simplify printer deployment across multiple locations.
